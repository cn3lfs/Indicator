# 简介

缠论可视化交易插件。在通达信里把缠师原文的 **分型 → 笔 → 线段 → 中枢 → 走势类型 → 三类买卖点**
精确画出，并叠加 **动力学（MACD 背驰）** 增强的信号质量与买卖点研判。所有定义、定理均贴近
《缠中说禅教你炒股票108课》原文，注释中标注课文编号可溯源。

# 功能介绍

插件以 14 个导出函数提供以下能力，可按需在通达信公式中组合（示例见文末）。

## 形态学（几何）

- **分型 / 笔 / 线段**：对原始 K 线做包含处理后识别顶底分型，连成笔，再划分线段（第62-71课）。
- **中枢**：由连续三段走势的重叠区间构造中枢，记录重叠区间 `ZG/ZD` 与全幅极值 `GG/DD`（第17/18课）。
- **走势类型**：把相邻中枢归并为 盘整 / 上涨 / 下跌（第17课）。

## 买卖点（形态学 + 动力学）

- **三类买卖点**：第一类（趋势背驰）、第二类（一买后回抽）、第三类（中枢突破回试不破）（第20/21课）。
- **信号质量**：价格背驰之外再用 MACD 柱面积确认，区分 观察 / 确认 / 标准背驰（第24/27课）。

## 进阶研判（动力学）

- **中枢关系**：相邻中枢按中心定理二判为 上涨延续 / 下跌延续 / 中枢扩展（第20课）。
- **背驰-转折**：一类买卖点后首段反弹力度，区分 中枢扩展 / 更大盘整 / 反趋势（第29课）。
- **三买后续**：三类买卖点之后中枢 扩张 还是 新生（趋势）（第21课）。
- **背驰段**：标出构成背驰的那段走势，作为区间套精确入场的观察区间（第27课）。

## 均线系统（第11-15课）

- **体位与吻**：短长均线的 女上位/男上位 体位，以及 飞吻/唇吻/湿吻 三类吻（反抗强度递增）。
- **均线趋势力度**：短长均线相交面积——这是第15课对「趋势力度/背驰」的原文原始定义（MACD 仅为第24课的辅助）。

# 安装方法

目前仅支持通达信软件使用。安装时，首先将CZSC.dll复制到通达信安装目录下的T0002\dlls目录之中，并在通达信公式管理器中将本dll加载到1号dll插件之中。

# 编译方法

通达信插件需要 32 位 Windows DLL。推荐在 WSL2 中使用 MinGW-w64 交叉编译：

```bash
sudo apt update
sudo apt install make g++ mingw-w64 gcc-mingw-w64-i686 g++-mingw-w64-i686 binutils-mingw-w64-i686
make mingw32
```

也可以直接运行仓库脚本安装工具链：

```bash
sh scripts/bootstrap-wsl-mingw.sh
```

运行核心回归测试：

```bash
make mingw32-test
```

在 WSL2 中，`make mingw32-test` 只适合有 Wine 或 Windows 执行环境时使用，因为它产出的是 Windows PE 可执行文件。常规回归测试使用 WSL 原生 `g++`：

```bash
make test
```

检查工具链、运行测试并构建 DLL：

```bash
sh scripts/build-mingw32.sh
```

# 示例指标公式

通达信公式中用 `TDXDLL1(编号, ...)` 调用本插件函数。线段点信号 `DLL` 是中枢/买卖点类函数的输入，
故每个指标都先取 `DLL`。两个示例均可直接粘贴为新指标使用，按需裁剪。

## 示例一：缠论主图（线段 + 中枢 + 三类买卖点 + 信号质量 + 背驰段）

加为主图叠加指标。

```text
DLL:=TDXDLL1(1,H,L,5);              { 笔级别线段点 }
SEG:=TDXDLL1(9,H,L,5);              { 线段级别转折点 }
HIB:=TDXDLL1(2,DLL,H,L);           { 中枢上沿 ZG }
LOB:=TDXDLL1(3,DLL,H,L);           { 中枢下沿 ZD }
SIG:=TDXDLL1(4,DLL,H,L);           { 中枢起止标记 }
BSP:=TDXDLL1(5,DLL,H,L);           { 三类买卖点 }
QLT:=TDXDLL1(10,DLL,H,L);          { 信号质量 0/1/2 }
SLP:=TDXDLL1(8,DLL,H,L);           { 线段斜率 }
BSG:=TDXDLL1(14,DLL,H,L);          { 背驰段端点 }
{ 中枢上下沿与起止竖线 }
IF(HIB,HIB,DRAWNULL), COLORYELLOW;
IF(LOB,LOB,DRAWNULL), COLORYELLOW;
STICKLINE(SIG,LOB,HIB,0,0), COLORYELLOW;
{ 笔(黄)与线段(红)连线 }
DRAWLINE(DLL=-1,L,DLL=+1,H,0), COLORYELLOW;
DRAWLINE(DLL=+1,H,DLL=-1,L,0), COLORYELLOW;
DRAWLINE(SEG=-1,L,SEG=+1,H,0), COLORRED;
DRAWLINE(SEG=+1,H,SEG=-1,L,0), COLORRED;
{ 笔端点的斜率数字 }
DRAWNUMBER(DLL=+1,H,SLP), COLORYELLOW, DRAWABOVE;
DRAWNUMBER(DLL=-1,L,SLP), COLORYELLOW;
{ 三类买卖点 }
BUY(BSP=1,LOW);
BUY(BSP=3,LOW);
SELL(BSP=12,HIGH);
BUYSHORT(BSP=2,LOW);
SELLSHORT(BSP=11,HIGH);
SELLSHORT(BSP=13,HIGH);
{ 标准背驰(质量2)的最强买卖点重点标注 }
DRAWICON(BSP>0 AND QLT=2,LOW,1);
{ 背驰段连线(洋红)，作为区间套入场观察区间 }
DRAWLINE(BSG=1,H,BSG=2,L,0), COLORMAGENTA;
DRAWLINE(BSG=-1,L,BSG=-2,H,0), COLORMAGENTA;
```

## 示例二：进阶研判标记（中枢关系 / 背驰-转折 / 三买后续）

可单独加为副图，或追加到示例一之后叠加显示。

```text
DLL:=TDXDLL1(1,H,L,5);
EXT:=TDXDLL1(11,DLL,H,L);          { 相邻中枢关系 }
REV:=TDXDLL1(12,DLL,H,L);          { 一类买卖点背驰-转折强度 }
AFT:=TDXDLL1(13,DLL,H,L);          { 三类买卖点后续 }
{ 中枢关系：2 扩展(下沿)、1 上涨(上沿)、-1 下跌(下沿) }
DRAWICON(EXT=2,L,2);
DRAWICON(EXT=1,H,8);
DRAWICON(EXT=-1,L,9);
{ 背驰-转折强度：1 中枢扩展 / 2 更大盘整 / 3 反趋势 }
DRAWNUMBER(REV>0,LOW,REV);
{ 三买后续：1 中枢扩张、2 中枢新生 }
DRAWICON(AFT=1,LOW,1);
DRAWICON(AFT=2,LOW,2);
{ 即时背驰预警：当下末段未走完即预警 1 见顶 / -1 见底 }
WAR:=TDXDLL1(17,DLL,H,L);
DRAWICON(WAR=-1,LOW,1);
DRAWICON(WAR=1,HIGH,2);
```

## 示例三：均线系统（体位 / 趋势力度 / 吻）

加为副图指标。均线差的正负即体位、柱高即趋势力度，前后同向柱的高度对比即「均线背驰」（第15课原文力度定义）。

```text
MAD:=TDXDLL1(15,H,L,5);            { 短长均线差(短-长) }
KIS:=TDXDLL1(16,H,L,5);            { 均线吻类型 }
{ 体位与力度：差值为正(女上位)红柱、为负(男上位)绿柱，柱高即趋势力度 }
STICKLINE(MAD>0,0,MAD,2,0), COLORRED;
STICKLINE(MAD<0,0,MAD,2,0), COLORGREEN;
{ 吻：1 飞吻 2 唇吻 3 湿吻(转折多由此始) }
DRAWICON(KIS=1,0,1);
DRAWICON(KIS=2,0,2);
DRAWICON(KIS=3,0,3);
```

## 标记含义说明

- **信号质量 `QLT`（10）**：`0` 观察、`1` 确认、`2` 标准背驰。`2` 表示价格背驰之外还得到
  MACD 柱面积缩小的动力学确认（第24课标准背驰），是最值得关注的买卖点。
- **中枢关系 `EXT`（11）**：按第20课走势中枢中心定理二在后一个中枢起点标记 —— `2` 中枢扩展
  （前后中枢全幅区间重叠，形成高级别中枢，对应第29课最后中枢扩展、第21课三买后中枢扩张）、
  `1` 上涨延续、`-1` 下跌延续。
- **背驰-转折 `REV`（12）**：按第29课对每个第一类买卖点标注其后首段反弹力度 —— `1` 最后中枢扩展
  （最弱反弹）、`2` 更大级别盘整、`3` 反趋势（最强），`0` 表示非一类买卖点。
- **三买后续 `AFT`（13）**：按第21课对第三类买卖点标注后续 —— `1` 中枢扩张（与后续中枢形成更大
  级别中枢）、`2` 中枢新生（形成同向新中枢/趋势），`0` 表示后续中枢尚未形成。注：单级别图上后续
  中枢常与三买共享枢轴，正例较少，多级别/真实行情中更易出现。
- **背驰段 `BSG`（14）**：按第27课标记第一类买卖点「最后那段构成背驰的走势」—— 买点起点 `1`、
  终点 `2`，卖点起点 `-1`、终点 `-2`，连成线即背驰段，可作为区间套精确入场的观察区间。
- **均线差 `MAD`（15）**：短均线(5)减长均线(20)，价格用 `(H+L)/2`。`>0` 女上位（多头）、`<0` 男上位
  （空头）；幅度即第15课原文的「趋势力度」（短长均线相交面积的瞬时值），前后同向力度走弱即均线背驰。
- **均线吻 `KIS`（16）**：第11课吻的三分类 —— `1` 飞吻（短均线略走平即继续，反抗最弱）、`2` 唇吻
  （贴近长均线但不破）、`3` 湿吻（升破/跌破长均线，反抗最强，转折多由湿吻开始）。
- **即时背驰预警 `WAR`（17）**：第15课「趋势平均力度」即时版——把最后一个线段点到当下视作未走完的末段，
  其力度（面积/时间）若比上一同向完成段弱且已创新极值，则在转折确认前预警：`1` 见顶、`-1` 见底、`0` 无。
- **笔·新笔标准（18）**：与 1 号笔输出相同（端点 ±1），但按缠论新笔标准用**合并 K 线**间隔（两分型间至少 1 根
  独立合并 K 线）判定，1 号用原始 K 线间隔。二者并存，可在示例一里把 `DLL:=TDXDLL1(1,H,L,5)` 换成
  `DLL:=TDXDLL1(18,H,L,5)` 对比。
- **线段·特征序列法（19）**：与 9 号线段输出相同（端点 ±1），但按第67课**特征序列**（逆向笔为元素、非包含
  处理后找顶/底分型）划分，9 号用保护点启发式。二者并存，可把示例一里 `SEG:=TDXDLL1(9,H,L,5)` 换成
  `SEG:=TDXDLL1(19,H,L,5)` 对比。注：无缺口情况按原文在分型处结束；有缺口情况当前作简化（也在分型处确认），
  尚未实现反向特征序列分型的二次确认。
