# 简介

缠论可视化交易插件。在通达信里把缠师原文的 **分型 → 笔 → 线段 → 中枢 → 走势类型 → 三类买卖点**
精确画出，并叠加 **动力学（MACD 背驰）** 增强的信号质量与买卖点研判。所有定义、定理均贴近
《缠中说禅教你炒股票108课》原文，注释中标注课文编号可溯源。

# 功能介绍

插件以 14 个导出函数提供以下能力，可按需在通达信公式中组合（示例见文末）。

## 形态学（几何）

- **分型 / 笔 / 线段**：对原始 K 线做包含处理后识别顶底分型，连成笔，再划分线段（第62-71课）。
- **中枢**：由连续三段走势的重叠区间构造中枢，记录重叠区间 `ZG/ZD` 与全幅极值 `GG/DD`（第17/18课）。
- **走势类型**：把相邻中枢归并为 盘整 / 上涨 / 下跌（第17课）。

## 买卖点（形态学 + 动力学）

- **三类买卖点**：第一类（趋势背驰）、第二类（一买后回抽）、第三类（中枢突破回试不破）（第20/21课）。
- **信号质量**：价格背驰之外再用 MACD 柱面积确认，区分 观察 / 确认 / 标准背驰（第24/27课）。

## 进阶研判（动力学）

- **中枢关系**：相邻中枢按中心定理二判为 上涨延续 / 下跌延续 / 中枢扩展（第20课）。
- **背驰-转折**：一类买卖点后首段反弹力度，区分 中枢扩展 / 更大盘整 / 反趋势（第29课）。
- **三买后续**：三类买卖点之后中枢 扩张 还是 新生（趋势）（第21课）。
- **背驰段**：标出构成背驰的那段走势，作为区间套精确入场的观察区间（第27课）。

## 均线系统（第11-15课）

- **体位与吻**：短长均线的 女上位/男上位 体位，以及 飞吻/唇吻/湿吻 三类吻（反抗强度递增）。
- **均线趋势力度**：短长均线相交面积——这是第15课对「趋势力度/背驰」的原文原始定义（MACD 仅为第24课的辅助）。

# 安装方法

目前仅支持通达信软件使用。**按你的通达信位数选择对应 DLL**（位数不匹配会加载失败）：

| 通达信版本 | 复制哪个文件 |
|------------|--------------|
| 32 位（经典版） | `build/CZSC.dll` |
| 64 位（新版） | `build/CZSC64.dll` |

安装步骤：

1. 将对应的 DLL 复制到通达信安装目录下的 `T0002\dlls` 目录之中；
2. 在通达信「功能 → 公式系统 → 公式管理器」里把本 dll 加载到 **1 号 dll 插件**（公式中即用 `TDXDLL1(编号,...)` 调用）；
3. 新建技术指标，粘贴下方示例公式即可。

发布产物统一放在 **`build/`** 目录。两个 DLL 均已**静态链接** MinGW 运行时（`libgcc`/`libstdc++`），自包含，
无需在通达信机器另装任何运行库——仅依赖系统自带的 `KERNEL32.dll`/`msvcrt.dll`。导出函数与公式用法两版完全一致。

# 配置（笔类型 / 笔结束 / 中枢构件）

20 号函数 `TDXDLL1(20,H,L,配置码)` 用**一个数字码**统一配置整条流水线，输出的端点信号可直接当作
`DLL` 喂给 2-5 号函数，从而得到对应的笔中枢或线段中枢。配置码按十进制位拆分：

| 位 | 含义 | 0（默认） | 1 |
|----|------|-----------|---|
| 个位 | 笔类型 | 严格笔（处理后≥5根合并K线） | 新笔（处理后≥4根合并K线且处理前≥5根原始K线） |
| 十位 | 笔结束 | 严格最高/最低点 | 允许次高/次低点 |
| 百位 | 中枢构件 | 笔中枢（笔端点构成中枢） | 线段中枢（线段端点构成中枢） |

配置码新增**千位=线段法**（`0` 保护点启发式 / `1` 第67课特征序列法，仅线段中枢时生效）。
例：`0`=严格笔+笔中枢（等价于 1 号笔）；`100`=线段中枢（等价于 9 号线段）；`1100`=线段中枢+特征序列法
（等价于 19 号）；`11`=新笔+允许次高。把示例一里 `DLL:=TDXDLL1(1,H,L,5);` 换成
`DLL:=TDXDLL1(20,H,L,101);` 即可让其后的中枢、买卖点全部按该配置计算。原 1/9/18/19 号函数保持不变。

## 30 号 mode 统一入口（推荐）

30 号把**配置与输出合并为一个 mode 码**：`mode = 配置码*1000 + 输出类型*10`。内部一次算成从笔到买卖点的
全部结果（带缓存），再按输出类型投影——不必再先 `DLL:=TDXDLL1(1..)` 产端点、逐个喂给 2-5 号。
输出类型：`0` 端点、`1` 中枢上沿、`2` 下沿、`3` 起止、`4` 三类买卖点、`5` 信号质量、`6` 中枢关系、
`7` 背驰-转折、`8` 三买后续、`9` 背驰段、`10` 均线差、`11` 均线吻、`12` 即时背驰、
`13` 带量校验的吻（湿吻放量 `4`=骗线嫌疑，需先用 40 号注册成交量）。

例（默认配置 0，笔中枢）：`HIB:=TDXDLL1(30,H,L,10);`（中枢上沿）、`BSP:=TDXDLL1(30,H,L,40);`（三类买卖点）。
若要线段中枢+特征序列法（配置码 1100）的三类买卖点：`TDXDLL1(30,H,L,1100040)`。旧 Func1-20 全部保留兼容。

## 40 号 旁路注册真实收盘价 / 成交量

通达信 DLL 计算函数只有 3 个输入槽，已被 `H/L`（及配置/mode 码）占满，因此 MACD 背驰与均线默认用
`(H+L)/2` 作收盘价代理。40 号提供**旁路注册**绕过该限制：

```tdx
XC:=TDXDLL1(40,C,V,0);   { 注册真实收盘价 C 与成交量 V，输出透传 C 便于核对 }
```

把这一行**置于其它本插件函数之前**，之后中枢/买卖点/均线/背驰等会在内容校验通过时自动改用真实 C，
湿吻放量骗线过滤（30 号输出 13）也才有成交量可用。校验规则：注册的收盘价每根都须落在 `[L,H]` 内
（真实 OHLC 必然如此），否则视为错配（如跨股票/过期数据）而**回落到 `(H+L)/2` 代理**——故未注册或注册
错误均不会算错，只是退化为现状（零回归）。注意通达信按公式行序求值，40 号必须排在消费函数之前。

# 示例指标公式

通达信公式中用 `TDXDLL1(编号, ...)` 调用本插件函数。线段点信号 `DLL` 是中枢/买卖点类函数的输入，
故每个指标都先取 `DLL`。下列示例均可直接粘贴为新指标使用，按需裁剪。

> **建议**：凡用到信号质量(10)、背驰段(14)、背驰-转折(12)、即时背驰(17)、均线/吻(15/16/30 输出 13)
> 的指标，第一行先写 `XC:=TDXDLL1(40,C,V,0);` 注册真实收盘价与成交量，使 MACD 背驰与均线用真实 C
> 而非 `(H+L)/2` 代理（不写则自动回落代理，结果仍有效）。

## 示例一：缠论主图（线段 + 中枢 + 三类买卖点 + 信号质量 + 背驰段）

加为主图叠加指标。

```text
XC:=TDXDLL1(40,C,V,0);             { 注册真实收盘价/成交量，使下方 MACD 质量与背驰用真实 C }
DLL:=TDXDLL1(1,H,L,5);             { 笔级别线段点 }
SEG:=TDXDLL1(9,H,L,5);             { 线段级别转折点 }
HIB:=TDXDLL1(2,DLL,H,L);           { 中枢上沿 ZG }
LOB:=TDXDLL1(3,DLL,H,L);           { 中枢下沿 ZD }
SIG:=TDXDLL1(4,DLL,H,L);           { 中枢起止标记 }
BSP:=TDXDLL1(5,DLL,H,L);           { 三类买卖点 }
QLT:=TDXDLL1(10,DLL,H,L);          { 信号质量 0/1/2 }
SLP:=TDXDLL1(8,DLL,H,L);           { 线段斜率 }
BSG:=TDXDLL1(14,DLL,H,L);          { 背驰段端点 }
{ 中枢上下沿与起止竖线 }
IF(HIB,HIB,DRAWNULL), COLORYELLOW;
IF(LOB,LOB,DRAWNULL), COLORYELLOW;
STICKLINE(SIG,LOB,HIB,0,0), COLORYELLOW;
{ 笔(黄)与线段(红)连线 }
DRAWLINE(DLL=-1,L,DLL=+1,H,0), COLORYELLOW;
DRAWLINE(DLL=+1,H,DLL=-1,L,0), COLORYELLOW;
DRAWLINE(SEG=-1,L,SEG=+1,H,0), COLORRED;
DRAWLINE(SEG=+1,H,SEG=-1,L,0), COLORRED;
{ 笔端点的斜率数字 }
DRAWNUMBER(DLL=+1,H,SLP), COLORYELLOW, DRAWABOVE;
DRAWNUMBER(DLL=-1,L,SLP), COLORYELLOW;
{ 三类买卖点 }
BUY(BSP=1,LOW);
BUY(BSP=3,LOW);
SELL(BSP=12,HIGH);
BUYSHORT(BSP=2,LOW);
SELLSHORT(BSP=11,HIGH);
SELLSHORT(BSP=13,HIGH);

DRAWICON(BSP>0 AND QLT=2,LOW,1);
{ 背驰段连线(洋红)，作为区间套入场观察区间 }
DRAWLINE(BSG=1,H,BSG=2,L,0), COLORMAGENTA;
DRAWLINE(BSG=-1,L,BSG=-2,H,0), COLORMAGENTA;
```

## 示例四：30 号统一入口（推荐写法，一次算成全链路）

30 号把配置与输出合并为一个 `mode = 配置码*1000 + 输出类型*10` 码，内部一次算成从笔到买卖点的全部结果
（带缓存），各输出共享同一次计算，无需先产端点再逐个喂下游。下例用**默认配置（笔中枢）**，把主图常用项
全部用 30 号取出：

```text
XC:=TDXDLL1(40,C,V,0);             { 注册真实 C/V，后续 30 号的质量/背驰用真实收盘价 }
PT :=TDXDLL1(30,H,L,0);            { 0  端点(笔)：顶 +1 / 底 -1 }
ZG :=TDXDLL1(30,H,L,10);           { 1  中枢上沿 ZG }
ZD :=TDXDLL1(30,H,L,20);           { 2  中枢下沿 ZD }
BSP:=TDXDLL1(30,H,L,40);           { 4  三类买卖点：买 1/2/3、卖 11/12/13 }
QLT:=TDXDLL1(30,H,L,50);           { 5  信号质量 0/1/2 }
{ 中枢上下沿 }
IF(ZG,ZG,DRAWNULL), COLORYELLOW;
IF(ZD,ZD,DRAWNULL), COLORYELLOW;
{ 笔连线 }
DRAWLINE(PT=-1,L,PT=+1,H,0), COLORYELLOW;
DRAWLINE(PT=+1,H,PT=-1,L,0), COLORYELLOW;
{ 三类买卖点 + 标准背驰(质量2)高亮 }
BUY(BSP=1 OR BSP=3,LOW);
SELLSHORT(BSP=11 OR BSP=13,HIGH);
DRAWICON(BSP>0 AND QLT=2,LOW,1);
```

> 想换中枢构件/笔类型/线段法，只改配置码即可：如**线段中枢+特征序列法**的三类买卖点为
> `TDXDLL1(30,H,L,1100040)`（配置码 `1100`*1000 + 输出 `4`*10）。配置码各位含义见上文「配置」一节。

## 标记含义说明

- **信号质量 `QLT`（10）**：`0` 观察、`1` 确认、`2` 标准背驰。`2` 表示价格背驰之外还得到
  MACD 柱面积缩小的动力学确认（第24课标准背驰），是最值得关注的买卖点。
- **中枢关系 `EXT`（11）**：按第20课走势中枢中心定理二在后一个中枢起点标记 —— `2` 中枢扩展
  （前后中枢全幅区间重叠，形成高级别中枢，对应第29课最后中枢扩展、第21课三买后中枢扩张）、
  `1` 上涨延续、`-1` 下跌延续。
- **背驰-转折 `REV`（12）**：按第29课对每个第一类买卖点标注其后首段反弹力度 —— `1` 最后中枢扩展
  （最弱反弹）、`2` 更大级别盘整、`3` 反趋势（最强），`0` 表示非一类买卖点。
- **三买后续 `AFT`（13）**：按第21课对第三类买卖点标注后续 —— `1` 中枢扩张（与后续中枢形成更大
  级别中枢）、`2` 中枢新生（形成同向新中枢/趋势），`0` 表示后续中枢尚未形成。注：单级别图上后续
  中枢常与三买共享枢轴，正例较少，多级别/真实行情中更易出现。
- **背驰段 `BSG`（14）**：按第27课标记第一类买卖点「最后那段构成背驰的走势」—— 买点起点 `1`、
  终点 `2`，卖点起点 `-1`、终点 `-2`，连成线即背驰段，可作为区间套精确入场的观察区间。
- **均线差 `MAD`（15）**：短均线(5)减长均线(20)，价格用 `(H+L)/2`。`>0` 女上位（多头）、`<0` 男上位
  （空头）；幅度即第15课原文的「趋势力度」（短长均线相交面积的瞬时值），前后同向力度走弱即均线背驰。
- **均线吻 `KIS`（16）**：第11课吻的三分类 —— `1` 飞吻（短均线略走平即继续，反抗最弱）、`2` 唇吻
  （贴近长均线但不破）、`3` 湿吻（升破/跌破长均线，反抗最强，转折多由湿吻开始）。
- **带量校验的吻（30 号输出 13）**：在 16 号纯价吻基础上用成交量校验湿吻（第12课）——湿吻处若近端
  成交量显著高于成交量均线（放量），多为诱多/诱空的骗线，单独标 `4`（骗线嫌疑），其余沿用 `1/2/3`。
  须先用 40 号注册真实成交量，否则无量退化为纯价吻分类。
- **即时背驰预警 `WAR`（17）**：第15课「趋势平均力度」即时版——把最后一个线段点到当下视作未走完的末段，
  其力度（面积/时间）若比上一同向完成段弱且已创新极值，则在转折确认前预警：`1` 见顶、`-1` 见底、`0` 无。
- **笔·新笔标准（18）**：与 1 号笔输出相同（端点 ±1），但按缠论新笔标准用**合并 K 线**间隔（两分型间至少 1 根
  独立合并 K 线）判定，1 号用原始 K 线间隔。二者并存，可在示例一里把 `DLL:=TDXDLL1(1,H,L,5)` 换成
  `DLL:=TDXDLL1(18,H,L,5)` 对比。
- **线段·特征序列法（19）**：与 9 号线段输出相同（端点 ±1），但落实第64/71课「线段只能被线段破坏」。
  9 号用保护点启发式（不保证此原则）。算法跟踪段内极值（上升:最高顶 / 下降:最低底），一旦从该极值起的
  **反向走势延伸出三笔、且第三笔破掉第一笔的结束位置**（第71课6.8，即上升线段见 lower high+lower low、
  下降线段见 higher low+higher high 的反向线段级别结构）→ 真破坏、线段在该极值处结束；单笔/两笔回抽
  （未破第一笔结束位置）不破坏线段，线段延伸到真正的最高/最低点。这样既不会被单笔破坏、也不会跳过中途
  成立的反向线段而把线段拉平。**推荐用 19 号画线段**，把示例里 `SEG:=TDXDLL1(9,H,L,5)` 换成
  `SEG:=TDXDLL1(19,H,L,5)` 即可。
