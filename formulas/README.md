# 缠论选股公式

需要 `build/CZSC.dll`（32 位）或 `build/CZSC64.dll`（64 位）放入通达信 `T0002/dlls/`，
并在公式管理器中绑定为 **1 号 DLL 插件**。

## 导入

通达信 → 功能 → 公式系统 → 公式管理器：

| 类型 | 文件 | 操作 |
|------|------|------|
| 条件选股 | `chan-first-buy.txt` | 新建 → 条件选股公式 → 粘贴内容 → 保存 |
| 条件选股 | `chan-second-buy.txt` | 同上 |
| 条件选股 | `chan-third-buy.txt` | 同上 |
| 主图指标 | `chan-all-buys.txt` | 新建 → 技术指标（主图叠加）→ 粘贴 → 保存 |

## 配置

默认使用笔中枢。如要线段中枢，把 `40` 换成 `1100040`（配置码 1100 × 1000 + 40）：

```tdx
BSP:=TDXDLL1(30,H,L,1100040);
```
