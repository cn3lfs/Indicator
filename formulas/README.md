# 缠论选股公式

基于 `CZSC.dll` 的 TDX 条件选股公式。

## 使用前提

1. 将 `build/CZSC.dll`（32 位通）或 `build/CZSC64.dll`（64 位通达信）放入通达信
   `T0002/dlls/` 目录
2. 在通达信「公式管理器 → DLL 插件」中绑定为 **1 号插件**
3. 确保通达信盘后数据完整（日线及以上周期）

## 条件选股公式

| 文件 | 用途 | 信号编码 |
|------|------|----------|
| `chan-first-buy.txt` | 第一类买点选股 | BSP=1 |
| `chan-second-buy.txt` | 第二类买点选股 | BSP=2 |
| `chan-third-buy.txt` | 第三类买点选股 | BSP=3 |

## 主图公式

| 文件 | 用途 |
|------|------|
| `chan-all-buys.txt` | 主图同时标注一买/二买/三买（图标） |

## 导入方法

1. 打开通达信 → 功能 → 公式系统 → 公式管理器
2. **选股公式**：新建 → 条件选股公式 → 粘贴对应 txt 内容 → 起名保存
3. **主图公式**：新建 → 技术指标公式（主图叠加）→ 粘贴 `chan-all-buys.txt`

## 配置切换

默认使用笔中枢（配置码 0）。如需线段中枢，把公式里的：

```
BSP:=TDXDLL1(30,H,L,40);
```

替换为（配置码见 `README.md` 主项目说明）：

```
BSP:=TDXDLL1(30,H,L,1100040);   { 线段中枢 + 特征序列法 }
```
