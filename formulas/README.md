# 缠论选股公式

需要 `build/CZSC.dll`（32 位）或 `build/CZSC64.dll`（64 位）放入通达信 `T0002/dlls/`，
并在公式管理器中绑定为 **1 号 DLL 插件**。

所有函数第三参数填 `0`（预留配置入口，当前统一使用默认配置：严格笔 + 严格极值 + 笔中枢）。

## 导入

通达信 → 功能 → 公式系统 → 公式管理器：

| 类型 | 文件 | 用途 |
|------|------|------|
| 条件选股 | `chan-first-buy.txt` | 日线 一买 |
| 条件选股 | `chan-second-buy.txt` | 日线 二买 |
| 条件选股 | `chan-third-buy.txt` | 日线 三买 |
| 条件选股 | `chan-multi-buy.txt` | 多级别共振：日线二买 + 30分二买/三买 |
| 主图指标 | `chan-all-buys.txt` | 笔+线段+三类买卖点+信号质量，完整主图 |
