# 缠论选股公式

需要 `build/CZSC.dll`（32 位）或 `build/CZSC64.dll`（64 位）放入通达信 `T0002/dlls/`，
并在公式管理器中绑定为 **1 号 DLL 插件**。

## 导入

通达信 → 功能 → 公式系统 → 公式管理器：

| 类型 | 文件 | 用途 |
|------|------|------|
| 条件选股 | `chan-first-buy.txt` | 日线 一买 |
| 条件选股 | `chan-second-buy.txt` | 日线 二买 |
| 条件选股 | `chan-third-buy.txt` | 日线 三买 |
| 条件选股 | `chan-multi-buy.txt` | 多级别共振：日线二买 + 30分二买/三买 |
| 主图指标 | `chan-all-buys.txt` | 日线主图 一买+二买+三买 图标标注 |

## 多级别共振说明

`chan-multi-buy.txt` 的逻辑：
1. 日线上近 5 天内出过二买
2. 30 分钟上近 2 天内（16 根 30 分 K）出过二买或三买
3. 两者同时满足 → 选中

> **DLL 限制**：`TDXDLL1(40,C,V,0)` 全局单槽注册 C/V。
> 30 分钟跨周期调用不复注 C/V，退化为 `(H+L)/2` 代理——信号检测不受影响，
> 仅 MACD 背驰质量判级精度略降。

## 配置

默认笔中枢。如需线段中枢，把 `40` 换成 `1100040`：

```tdx
BSP:=TDXDLL1(30,H,L,1100040);
```
